# Set a Twitter name
As you may have seen a lot of users have their ENS names as their Twitter names.

(Insert images of recognizable people's twitter profiles with .eth set here)

## How to set an ENS name as a Twitter name
Twitter does not yet integrate ENS names, or verify them, so setting them as your Twitter name is simpler than it first appears.
All you need to do is to edit your name and enter it in!

### Edit your Twitter profile
Go to [Twitter](https://www.twitter.com) and click on your `Profile`

<BrowserWindow url="twitter.com">
  <Image img="/img/howto/set-twitter-name-1.png"/>
</BrowserWindow>

Click the `Edit` button in the upper right of your profile page to edit it.

<BrowserWindow url="twitter.com">
  <Image img="/img/howto/set-twitter-name-2.png"/>
</BrowserWindow>

Type in your ENS name in the `Name` field

<BrowserWindow url="twitter.com">
  <Image img="/img/howto/set-twitter-name-3.png"/>
</BrowserWindow>

Press `Save` to save it.

<BrowserWindow url="twitter.com">
  <Image img="/img/howto/set-twitter-name-4.png"/>
</BrowserWindow>

That's it! You're done and should now be sporting your ENS name on Twitter!
