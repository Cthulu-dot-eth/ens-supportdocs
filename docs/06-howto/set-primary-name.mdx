# Set a Primary name

To set your ENS name as Primary in your wallet:

:::info Instructions
1. Make sure the `ETH Address` record points to the wallet address

1. Connect your wallet to the <Link to="https://app.ens.domains"><b>ENS Manager App</b></Link> and set the ENS name as `Primary`.
:::

# Set a Primary name 2
:::caution Pre-requisites
 - The `ETH Address` record of the ENS name must first point to the wallet you want to set the `Primary Name` in

If you haven't set the `ETH Address` record to your ENS name, see
 - <Link to="/docs/howto/add-wallet-address#eth-address"><b>How-To: </b>Add a Wallet Address <i>(ETH Address)</i></Link>
:::
