# The ETH Address record
The `ETH` Address record is one of the main records of your ENS name. As it not only affects where Ethereum funds sent to it ends up, but also provides the forward resolution required when setting up a `Primary name`.

:::info Add wallet address
To find out how to add a wallet address to your ENS name, see: 
- <Link to="/docs/howto/add-wallet-address"><b>How-To: Add a wallet address</b></Link>
:::
